# アプリ名：Mission Days

## 概要
このアプリは、毎日のルーチン作業の効率化、目標達成のやる気向上のため作成しました。  

## 機能一覧
- ユーザー登録
- Cookieを用いた自動ログイン機能
- タスク完了によるスタンプ獲得（今後追加予定の機能）
- スタンプカードコンプリートによるごほうびチケット獲得（今後追加予定の機能）
- タスクの追加・編集・削除（CRUD）※編集・削除は今後追加予定
- 完了済みタスクのアーカイブ（今後追加予定の機能）

## 使用技術
| カテゴリ | 技術スタック |
| :--- | :--- |
| 言語 / フレームワーク | PHP 8.5.x / Laravel 12.x |
| データベース | MySQL 8.4.x |
| インフラ | Docker (Laravel Sail) |
| デザイン | 自作アイコン・背景画像 |

## こだわったポイント
- アプリのような挙動になるよう、スライドやモーダル表示を取り入れました。
- 操作しやすいように、アイコンや色でUIを分かりやすくしています。
